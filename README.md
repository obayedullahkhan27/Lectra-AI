# AI Book Companion | Vapi, ElevenLabs

An AI-powered platform that lets users have real-time voice conversations with their books. Built with Next.js, Vapi, ElevenLabs, and MongoDB, this project transforms PDFs into interactive voice-enabled companions using natural voice synthesis and intelligent retrieval.

## ✨ Introduction

AI Book Companion is a voice-first platform where users can upload PDF books and interact with them through real-time AI conversations.

The system processes uploaded books, extracts content, creates intelligent embeddings, and allows users to ask questions, request summaries, and explore concepts using natural voice conversations.

With custom AI voice personas powered by ElevenLabs and real-time communication through Vapi, the platform creates an immersive reading experience beyond traditional books.

## 🔋 Features

### 👉 PDF Upload & Ingestion

Upload PDF books with automatic text extraction, intelligent chunking, and embedding generation for accurate context retrieval.

### 👉 Real-Time Voice Conversations

Talk naturally with your uploaded books using real-time low-latency voice conversations powered by Vapi.

### 👉 AI Voice Personas

Choose from multiple AI voice personalities and preview them instantly using ElevenLabs high-quality voice synthesis.

### 👉 Smart Summaries & Insights

Request chapter summaries, concept explanations, and deep-dive insights directly from your books.

### 👉 Live Session Transcripts

Every voice interaction is automatically converted into text transcripts for future reference.

### 👉 Library Management

Manage personal uploads and search through books using a fast and organized library system.

### 👉 Authentication & Subscription System

Secure user authentication with Clerk including email/social login and premium access management.
